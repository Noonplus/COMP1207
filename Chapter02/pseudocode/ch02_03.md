#### ตัวอย่างคำสั่งเทียมที่ 2.3 โคดเทียมโปรแกรมแบบหลายทางเลือกโดยใช้เงื่อนไข
```
BEGIN
    INPUT อายุ;
    IF อายุ >= 60 THEN
        OUTPUT “เป็นผู้สูงวัย”;
    ELSE IF อายุ >= 22 THEN
        OUTPUT “เป็นวัยทำงาน”;
    ELSE IF อายุ >= 15 THEN
        OUTPUT “เป็นวัยรุ่น”;
    ELSE
        OUTPUT “เป็นวัยเด็ก”;
    END IF
END
```
